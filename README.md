# Soalex Alpha

**Interner Alpha-Test-Kanal von Soalex.** Hier landet jeder grüne
`main`-Build automatisch — ungefiltert und schnelllebig.

> ⚠️ **Alpha, nicht Beta.** Dieser Kanal kann mehrmals täglich aktualisiert
> werden und ist nur für die aktive Entwicklung gedacht. Für stabilere
> Erprobung den Beta-Kanal nutzen: https://github.com/thealkly/SoalexBeta

Das Alpha-Add-on hat den Slug `soalex_alpha` und installiert **parallel**
zum Beta-Add-on (eigene Daten, eigene DB, eigene Lizenz). Immer nur **eines**
gleichzeitig starten — zwei Steuerungen auf derselben Hardware widersprechen
sich.

## Installation

1. In Home Assistant: **Einstellungen → Add-ons → Add-on-Store**.
2. Oben rechts **⋮ → Repositories**.
3. URL eintragen: `https://github.com/thealkly/SoalexAlpha`
4. Nach dem Refresh erscheint **Soalex (Alpha)** in der Add-on-Liste.
